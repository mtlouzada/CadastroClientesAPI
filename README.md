# CadastroClientesAPI

Aplicação CRUD utilizando C# e .NET Core 3.1

## Tecnologias utilizadas
- C#
- .NET Core 3.1

## Como executar
1. Clone o repositório
1. Abra o projeto no Visual Studio
1. Execute o projeto
1. Acesse a URL `https://localhost:5001/swagger/index.html`
1. Pronto, a aplicação estará rodando
